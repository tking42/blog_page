FlexBox

Step 1: Make a div a container in HTML

<div class="Container">
<div class="Box">Box1</div>
<div class="Box">Box2</div>
<div class="Box">Box3</div>
</div>

Step 2: Define a container as a flexbox

.container {
display: flex;
}

Step 3: Set flex container (default is row)

.container {
display: flex;
flex-direction: column;
}

Step 4: Add flex properties eg. justify-content and align-content

.container {
display: flex;
flex-direction: column;
justify-content: space-evenly;
align-items: center;
}

Other Properties:

Other properties include flex-wrap, gap, and flex-grow. Each property manipulates the alignment of the contents of the container along the main and cross axis of flexbox.
